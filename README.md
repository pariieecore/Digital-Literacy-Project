 Project Overview

This repository contains my Digital Literacy Project, created as part of the CSE0001 course. The aim of this project is to develop essential digital skills required for academic and professional growth.

The project covers important areas such as digital awareness, professional online presence, coding platforms, communication etiquette, and cybercrime awareness. Each task reflects practical learning and helps in understanding how to use digital tools responsibly and effectively.


 Task Summary

 
 Task 1: Digital Literacy Infographic

Created an infographic explaining digital literacy concepts using Canva, including safe practices and useful tools.

 Task 2: Digital Portfolio

Developed professional profiles on GitHub, LinkedIn, and Kaggle to build an online presence.

 Task 3: Platforms Exploration

Completed a beginner coding challenge and created a Digital Literacy Awareness Quiz using Google Forms.
